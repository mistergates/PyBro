# PyBro - A simple **Py**thon Web **Bro**wser

## Requirements
* Python 3+
* wxPython

## Usage
Clone the repo:
```
git clone https://github.com/mistergates/PyBro.git
```

Change into repo directory:
```
cd PyBro
```

Install requirements:
```
pip install -r requirements.txt
```

Launch browser:
```
python -m PyBro
```
